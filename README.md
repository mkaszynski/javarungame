## Star Runner

Fun infinite runner style game, made with javascript.

![run_game.js](screenshot.png)


### Installation

Clone this repository with:

```
git clone https://github.com/mkaszynski/javarungame
```

### Usage

With google, run with:

```
google-chrome index.html
```

Or, use online version:

```
https://mkaszynski.github.io/javarungame/
```

#### Controls

- press space bar or click screen to jump
